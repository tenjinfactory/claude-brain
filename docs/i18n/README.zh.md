## 问题

你在多台设备上使用 Claude Code。笔记本上 Claude 学会了你的编码习惯，台式机上有自定义技能，云服务器上有不同的规则。**它们之间互不通信。**

每次换设备，你都会丢失上下文。你得重新教 Claude 同样的东西。你精心编写的 CLAUDE.md 留在了另一台机器上。

## 解决方案

```
# 设备 A（工作笔记本）
> /brain-init git@github.com:you/my-brain.git
✓ 大脑已导出：42 条记忆，3 个技能，5 条规则
✓ 已推送到远程仓库

# 设备 B（家用台式机）
> /brain-join git@github.com:you/my-brain.git
✓ 已拉取大脑：42 条记忆，3 个技能，5 条规则
✓ 已与本地状态合并
✓ 自动同步已启用

# 搞定了。每次会话开始/结束都会自动同步。
# 你的大脑跟着你走。
```

两行命令。零日常操作。永远有效。

## 为什么选择 claude-brain？

| 工具 | 功能 | claude-brain 的优势 |
|------|------|-------------------|
| **claude-mem** | 增强单台设备的记忆 | 在所有设备之间同步**整个大脑** |
| **chezmoi / dotfiles** | 复制配置文件 | **智能合并**知识 — 解决矛盾、去重 |
| **手动复制 CLAUDE.md** | 能用但麻烦 | 每次会话开始/结束自动静默同步 |

**没有其他工具能做到 Claude Code 大脑状态的跨设备语义合并。**

## 核心特性

- **自动同步** — 每次会话开始/结束都会触发钩子，零操作
- **语义合并** — LLM 驱动的记忆和 CLAUDE.md 去重（不是简单覆盖）
- **N 路合并** — 笔记本 + 台式机 + 云服务器智能汇聚
- **加密** — 可选的 `age` 加密保护大脑快照
- **团队共享** — 与团队成员共享技能、Agent 和规则
- **自动进化** — 将稳定的模式从记忆提升为持久配置
- **安全优先** — 密钥自动剥离，环境变量排除，强制使用私有仓库
- **成本极低** — 通过 `claude -p` 调用，每月约 $0.50-2.00

## 快速开始

### 安装

```bash
# 从插件市场安装
/plugin marketplace add tenjinfactory/claude-brain
/plugin install claude-brain-sync
```

### 初始化（第一台设备）

```
/brain-init git@github.com:you/my-brain.git
```

### 加入（其他设备）

```
/brain-join git@github.com:you/my-brain.git
```

### 启用加密

```
/brain-init git@github.com:you/my-brain.git --encrypt
```

完成。自动同步会处理后续的一切。

## 命令列表

| 命令 | 说明 |
|------|------|
| `/brain-init <remote>` | 用 Git 远程仓库初始化大脑网络 |
| `/brain-join <remote>` | 加入已有的大脑网络 |
| `/brain-status` | 显示大脑清单和同步状态 |
| `/brain-sync` | 手动触发完整同步 |
| `/brain-sync --dry-run` | 预览将要同步的内容（不实际执行） |
| `/brain-evolve` | 将稳定模式从记忆提升为配置 |
| `/brain-conflicts` | 查看和解决合并冲突 |
| `/brain-share <类型> <名称>` | 与团队共享技能、Agent 或规则 |
| `/brain-shared-list` | 列出网络中所有共享的资源 |
| `/brain-log` | 显示同步历史 |

## 依赖

- `git` — 同步传输
- `jq` — JSON 处理（`apt install jq` / `brew install jq`）
- `claude` CLI — 语义合并（已随 Claude Code 安装）
- `age` — 可选，用于加密

## 安全说明

- **密钥永远不会导出** — OAuth 令牌、API 密钥、环境变量全部排除
- **基于模式的密钥扫描** — 检测到潜在密钥时发出警告
- **MCP 环境变量已剥离** — 服务器配置同步不包含凭据
- **强制私有仓库** — 检测到公开仓库时警告
- **自动备份** — 每次导入前自动创建备份
- **语义合并使用 Claude API** — 记忆内容会发送到 `claude -p`
- **信任网络中的所有设备** — 导入的技能以 Claude 的权限执行
